# calculator_vue

Este projeto é uma calculadora aritmética desenvolvida com Vue 3 e Vite.

## Funcionalidades

- Dois campos para inserir números
- Um select para escolher a operação
- Cálculo automático ao alterar valores
- Operações: soma, subtração, multiplicação e divisão
